This is an unknown file, added in 0.996, probably for security reasons.
Maybe the file contains Additional Signatures (A.S.) necessary for validation of the PUP

Offset Size Description
0x0 0x8 Magic (SCEAS\x00\x00\x01)
0xC 0x4 File Size
0x10 0x4 Unknown
0x14 0x4 Unknown
0x18 0x4 Unknown
0x1C 0x4 Zeros


This file is missing inside the DEX PUP v 0.945.040 (and all the pre-0.996 pups)
